# Alive State

[![Build Status](https://travis-ci.org/ogom/alive_state.png?branch=master)](https://travis-ci.org/ogom/alive_state)

Alive Application State on Rails.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'alive_state'
```

And then execute:

```
$ bundle
```
